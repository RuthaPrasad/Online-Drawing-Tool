Go Up, Down, Left or Right !

Ever wanted to draw something as part of the college assignment but never found the right tool to use? Found a tool but it could only be used for a single subject? If the answers to either of the above questions is yes then dont worry. Techie Doodles is the go to tool for all your needs.

Aim: We aim to design a drawing tool that is capable of drawing anything and everything based on engineering subjects. Whether its that electrical circuit or that mechanics problem with all the forces, the ER-diagram as part of your mini project or the logic circuit as part of the assignment. Techie Doodles will do it all.

Features:

  • Drag and drop implementation
  • Voice responsive tutorial
  • Save work as image
  • Completely customizable
  • Wide range of tools
  • Code away *
  • Prediction of components *
  • Make the correction process automated *

* To be implemented in the near future.
